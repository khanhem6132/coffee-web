# Coffee Web Application

ASP.NET MVC web application for managing spare parts products.

## Features
- Product listing and categories
- Shopping cart functionality
- Product management
- Responsive UI using HTML, CSS, JavaScript

## Technologies
- ASP.NET MVC
- C#
- Entity Framework
- SQL Server
- HTML / CSS / JavaScript

## Notes
This repository contains the source code of the project.
Database files are not included.
