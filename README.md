# Coffee Web Application

This project is a web application developed as part of a university course related to database systems.  
The application demonstrates how a web system can interact with a relational database to manage and display product information.

## Project Overview
The system allows users to browse coffee products, view product categories, and interact with data stored in a database.  
It focuses on database integration and basic CRUD operations within a web application environment.

## Main Features
- Display coffee products from the database
- Browse products by category
- Database-driven web pages
- Basic data management structure

## Technologies Used
- ASP.NET MVC
- C#
- Entity Framework
- SQL / Database systems
- HTML
- CSS
- JavaScript

## Purpose of the Project
The goal of this project is to demonstrate:
- Database design and integration
- Data retrieval and display in a web application
- Interaction between ASP.NET MVC and relational databases

## Notes
This repository contains the **source code of the application**.
